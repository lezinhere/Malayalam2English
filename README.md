====================================================
Malayalam → English Book Translator
Developed by Lezin
====================================================

This tool converts Malayalam DOCX files into English using AI.

----------------------------------------------------
SYSTEM REQUIREMENTS
----------------------------------------------------

1. Windows 10 or Windows 11
2. Internet connection (required first time only)
3. Python 3.10 or higher installed

Download Python from:
https://www.python.org/downloads/

IMPORTANT:
During installation, make sure to check:
[✓] Add Python to PATH


----------------------------------------------------
FIRST TIME SETUP (ONLY ONCE)
----------------------------------------------------

Step 1:
Install Python (if not already installed).

Step 2:
Open this folder.

Step 3:
Double-click "setup.bat"

Step 4:
Wait until installation completes.
(This may take several minutes.)

When you see:
"Setup completed successfully!"
You are ready to use the app.


----------------------------------------------------
HOW TO USE THE TRANSLATOR
----------------------------------------------------

Step 1:
Double-click "run.bat"

Step 2:
Your browser will open automatically.

Step 3:
Click "Upload DOCX file"

Step 4:
Select your Malayalam .docx file.

Step 5:
Click "Start Translation"

Step 6:
Wait until translation completes.

Step 7:
Click "Download Translated Document"


----------------------------------------------------
IMPORTANT NOTES
----------------------------------------------------

• First run requires internet connection to download the AI model (~1GB).
• After the first run, it works offline.
• Large documents may take several minutes to translate.
• Only DOCX files are supported.


----------------------------------------------------
IF SOMETHING DOES NOT WORK
----------------------------------------------------

1. Make sure Python is installed.
2. Make sure Python was added to PATH.
3. Close everything and run setup.bat again.
4. Ensure you have internet connection (first run only).


----------------------------------------------------
End of Instructions
